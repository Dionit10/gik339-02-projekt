# Books CRUD – GIK339

## Beskrivning
En CRUD Single Page Application byggd med Node.js, Express och SQLite.
Applikationen låter användaren skapa, läsa, uppdatera och ta bort böcker.

## Tekniker
- Node.js
- Express
- SQLite
- Vanilla JavaScript
- Bootstrap

## Starta projektet lokalt
1. Klona repot
2. Kör `npm install`
3. Kör `npm start`
4. Öppna http://localhost:3000
